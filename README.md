# 🐾 WooCommerce Pet Shop Website

## Περιγραφή

Δημιουργία ηλεκτρονικού καταστήματος για προϊόντα κατοικίδιων ζώων με χρήση WordPress και WooCommerce.

Το project περιλαμβάνει σχεδιασμό και παραμετροποίηση ιστοσελίδας, δημιουργία σελίδων προϊόντων και προσαρμογή της εμφάνισης με χρήση CSS.


## Χαρακτηριστικά

- Δημιουργία ηλεκτρονικού καταστήματος
- Διαχείριση προϊόντων και κατηγοριών
- Σελίδες προϊόντων
- Καλάθι αγορών
- Προσαρμοσμένο design με CSS
- Responsive σχεδίαση για κινητές συσκευές



## Τι υλοποίησα

- Παραμετροποίηση WordPress και WooCommerce
- Σχεδιασμό και βελτίωση του UI
- Προσαρμογή CSS για την εμφάνιση των σελίδων
- Οργάνωση περιεχομένου και δομής ιστοσελίδας
